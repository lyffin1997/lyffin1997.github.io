如何重写comparator比较器

  • 方法一：
//Arrays.sort(T[] a, Comparator<? super T> c)
//T[] a：需要比较的数组
//Comparator<? super T> c：比较器
Car[] arr = new Car[3];
Arrays.sort(arr, new Comparator<Car>() {
    @Override
    public int compare(Car o1, Car o2) {
        return o1.position - o2.position;
    }
});
  • 方法二：
Car[] arr = new Car[3];
Arrays.sort(arr, (a, b) -> {
    //返回值>0交换
    return b.position-a.position;
});
